# Project Overview

BrightTV’s CEO aims to increase the company’s subscription base during the current financial year. To support this objective, this case study analyzes user engagement, viewing patterns, and subscription trends to provide actionable insights for the Customer Value Management (CVM) team.

# Dataset Description

The dataset contains detailed information about BrightTV users, including:

User demographics: age, gender, province, and race.

Viewing behavior: channels watched, duration of viewership, and frequency.

Engagement metrics: total users, weekly watches, monthly engagement, and content preferences.


# Analysis Goals

User vs. Channel Engagement: Identify which channels and content types attract the most viewers.

User Demographics vs. Viewing Patterns: Understand which age groups, genders, or provinces show higher engagement.

Content Optimization: Highlight opportunities to create or promote content that drives higher consumption.

Subscription Growth Opportunities: Identify periods, channels, or demographics that could benefit from targeted subscription campaigns.


# Methodology / What Was Done

Data Preparation:

Standardized channel names and grouped minor or unclear channels as Miscellaneous/Unclassified Content.

Converted timestamps from UTC to South African Time (UTC+2).

Transformed duration from HH:MM:SS format into numeric minutes and created duration categories: Short, Medium, Long.

SQL Analysis:

Analysize watch duration and total users by channel, age, gender, race, province, duration category, week, month, and hour.

Visualizations (Excel):

Created graphs showing trends across demographics, duration categories, channels, and time (hourly, weekly, monthly).

Designed charts to highlight engagement patterns and actionable insights.

# Key Insights

Channel Popularity: Certain channels dominate viewership; minor or misnamed channels were grouped under Miscellaneous/Unclassified Content.

Duration Trends: Most users watch short durations, though some channels maintain higher engagement.

Demographic Patterns: Age and gender significantly influence content preferences.

Peak Viewing Times: Highest engagement occurs between 18:00–21:00, with weekends showing increased activity.

Monthly Trends: Viewership spikes in certain months, likely linked to content releases or marketing campaigns.

User Retention: The majority of users return multiple times per week, indicating strong ongoing engagement.
